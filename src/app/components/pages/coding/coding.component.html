Como crear tu propio servidor de forma sencilla

Levantar emulador y base de datos

SexyRo es un proyecto opensource basado en el emulador rAthena.
Podrás descargar los recursos necesarios para crear tu proyecto en los siguientes links:

Antes de comenzar es necesario que tengas instalado - en tu computador para levantar base de datos.
Instalado docker entra al proyecto y ubicado aqui debes ejecutar el comando

Creado y ejecutado el contenedor podrás ejecutar el servidor ya previamente configurado.
Ejecutar archivo llamado start.bat

Se abrirán las consolas correspondientes a map-server.exe, login-server.exe y char-server.exe.
¡Felicitaciones ya tienes tu emulador corriendo!

Conectar cliente con emulador

En el pack de herramientas encontrarás un cliente de prueba el cual funciona de forma local. Debes copiar los archivos dentro de la carpeta “Cliente de prueba” en la carpeta donde tengas Instalado

¡Ejecuta el archivo RagnarokRE_patched.exe y comienza a jugar!

Crea un cliente para tu servidor

Para la creación de un cliente personalizado para tu servidor encontrarñas un archivo comprimido que termina en eRagexeRE, debes extraer este archivo junto con el programa Nemo Master que encontrarás en el pack para crear tu servidor.
Ejecutado este proyecto encontrarás una ventana similar en donde podrás seleccionar las caracteristicas con la que quedará tu cliente, desde el nombre de tu servidor, estilos, entre otras cosas. Aquí cargarás el cliente limpio con el botón Browser

Dentro del pack encontrarás un archivo llamado sexy-ro.log el cual vienen ya pre-cargadas algunas configuraciones por defecto para el cliente y lo podrás cargar pinchando en el botón Load Profile. Listas las configuraciones para tu cliente debes pinchar en el botón Load Client.

¡Listo tu cliente, podrás utilizarlo en tu servidor!

Crea un grf para tu servidor

Para crear tu popio grf encontrarás un programa llamado GRF Editor el cual debes instalarlo en tu computador y al ejecutarlo encontrarás una ventana como esta.

Puedes seleccionar el archivo custom.grf que viene en el pack y modificarlo a gusto.
Considera modificar el archivo sclienteinfo.xml en donde deberás poner la IP de tu servidor en la etiqueta address, además el nombre de tu servidor o las imagenes de carga
